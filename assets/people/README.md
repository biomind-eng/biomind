# People headshots

Drop faculty / student / alumni headshots here. Recommended:

- Square crop (~400 × 400 px)
- JPEG, <120 KB after compression
- Filename: `firstname-lastname.jpg`

Reference from a profile card by replacing the initials placeholder with an image, e.g.:

```markdown
::: {.bm-person-photo}
![](../assets/people/firstname-lastname.jpg)
:::
```
