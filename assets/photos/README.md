# Photos

Drop cluster-wide photos here (events, group photos, lab spaces, etc.). Reference them from `.qmd` pages as relative paths, e.g. `![](../assets/photos/group-2026.jpg)`.

Recommended:
- JPEG, optimized for web (<300 KB ideal)
- Landscape ~1600 × 900 for hero/cover images
- Always include alt text in the Markdown reference
